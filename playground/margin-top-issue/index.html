When a 'div' inside another 'div' has set a margin-top, the margin-top appears to be in the parent 'div'

if you inspect the element, the margin is applied to the desire element but the result is like the margin would have being applied to the parent 'div'


In the present example there are 2 similar situations:

'section' has a margin-top: 100px but it is like that margin-top is set to his parent: 'main'
'h1' also has a margin-top: 90px but is applied to his parent: 'section'

A workaround is just adding a 'overflow: hidden' to the parent container also, when the parent container ('main' or 'section' in this example) has some content like text, it's no necessary using 'overflow: hidden'
